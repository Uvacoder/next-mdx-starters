How to use [degit][degit] to clone a repository or path from GitHub to scaffold a project.

### Bring in a folder

```bash
$ npx degit github:user/repo[/subdirectory] <path>
```

### Bring in a branch

```bash
$ npx degit github:user/repo#branch <path>
```

[degit]: https://github.com/Rich-Harris/degit
